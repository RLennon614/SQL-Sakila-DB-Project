üìä Key Analysis Areas (Sakila Schema)

My scripts address common business challenges faced by the Sakila rental store:

Inventory Optimization: Identifying overstocked/understocked films and tracking rental rates by category.

Customer Lifetime Value (CLV): Calculating the total rental revenue generated by specific customers.

Revenue Performance: Analyzing daily, monthly, and quarterly sales trends using complex date functions.

Ranking and Segmentation: Using window functions to rank films by popularity within their respective categories and segmenting customers based on rental frequency.

A Stored Procedure that accepts a film ID and returns the title, rental rate, and the count of available copies across all stores.
